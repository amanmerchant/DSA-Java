# DSA in Java

This repository contains my practice of Data Structures and Algorithms implemented in Java.
It is focused on improving problem-solving skills, understanding core concepts, and preparing for technical interviews as a fresher.

## Topics Covered
- Arrays
- Strings
- Recursion
- Linked List
- Stack & Queue
- Trees
- Graphs
- Dynamic Programming

## Structure
Each folder contains Java files with:
- Problem statement
- Approach / logic
- Time and space complexity

## Goal
To build strong DSA fundamentals and consistent problem-solving habits for software engineering interviews.
